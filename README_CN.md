# AirCanvas 用户指南

AirCanvas 是一个基于手势识别的隔空绘图工具，让你无需接触屏幕即可在空中书写、演示。

## 🎮 快速上手

1. **运行程序**：
   ```bash
   python main.py
   ```
2. **基本操作**：
   - **绘图/操作**：统一使用**捏合手势**（拇指+食指）来执行当前选中的工具动作。
   - **快速切换工具**：✨ **食指单指上/下滑动**（其他手指弯曲）快速切换工具，无需点侧边栏！
   - **侧边栏选择**：悬停在按钮上停留 0.4 秒自动选中，或捏合确认。
   - **显示/隐藏 UI**：双指（食指+中指）伸出，其他手指弯曲。
   - **撤销/重做**：三指滑动（向左撤销，向右重做）。

## 🛠️ 工具模式 (NEW)

不再自动切换模式，避免误触。有两种方式选择工具：

### ✨ 手势快捷切换（推荐）
- **食指单指向上滑动**：切换上一个工具  
- **食指单指向下滑动**：切换下一个工具  
- 手始终在画面中心，检测最稳定！

### 侧边栏选择（辅助）
- 悬停按钮 0.4 秒自动选中  
- 或捏合确认选择  
- 侧边栏已移到靠内位置，避开边缘盲区

### 三大工具

1. **🖊️ Pen (画笔)**
   - 捏合：绘制线条
   - 粒子特效：按 `1` 开启
   - 钢笔效果：按 `p` 开启（速度感压）

2. **🧽 Eraser (橡皮)**
   - 捏合：擦除经过的区域
   - 光标：显示为大圆圈

3. **🔦 Laser (激光笔)**
   - 光标：始终跟随食指移动
   - 捏合：绘制消失墨水（GoodNotes 风格，1.5秒后消失）

## ✨ 视觉特效

- **水波纹**：捏合开始、点击按钮时触发。
- **彩虹笔**：右侧笔刷栏选择 `Rainbow`，颜色随时间流转。
- **直线辅助**：按 `l` 开启，自动将手绘线变直。

## ⌨️ 快捷键列表

| 按键 | 功能 | 说明 |
|------|------|------|
| `t` | 切换工具 | 循环切换 Pen / Eraser / Laser |
| `q` | 退出 | 退出程序 |
| `c` | 清空 | 清空画布 |
| `s` | 保存 | 保存当前画作到 captures 文件夹 |
| `z` | 撤销 | 撤销上一步操作 |
| `y` | 重做 | 重做上一步操作 |
| `h` | 帮助 | 显示/隐藏帮助界面 |
| `p` | 钢笔效果 | 开启/关闭速度感压效果 |
| `l` | 直线辅助 | 开启/关闭直线自动美化 |
| `b` | 切换笔刷 | 切换实线/虚线/发光/彩虹笔 |
| `[` / `]` | 切换颜色 | 上一个/下一个颜色 |
| `-` / `+` | 调整粗细 | 减小/增加笔刷粗细 |
| `1` | 粒子特效 | 开关粒子拖尾 |
| `2` | 激光笔开关 | (全局开关) 是否允许激光笔显示 |
| `3` | 掌心HUD | 开关悬停信息面板 |
